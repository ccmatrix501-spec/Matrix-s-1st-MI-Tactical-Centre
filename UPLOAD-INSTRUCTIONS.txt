1st M.I. Tactical Centre mobile web build v1.3.9

1. Extract this ZIP.
2. Copy every extracted file and folder into your cloned GitHub repository folder.
3. Choose Replace files when Windows asks.
4. In GitHub Desktop, commit with: Fix mobile modifier menu and header
5. Push origin.
6. Wait for GitHub Pages to finish deploying.
7. Open the site on your phone. The first v1.3.9 load clears the old cached version and reloads once automatically.
8. Confirm Menu -> Quick Actions says "Mobile web build v1.3.9".

The modifier window now has a fixed Done × button in the top-right and a Done button at the bottom.
